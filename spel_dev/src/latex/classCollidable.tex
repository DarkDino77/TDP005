\doxysubsection{Detailed Description}
A collidable object in the game that inherits from the \mbox{\hyperlink{classUpdatable}{Updatable}} class. 

The \mbox{\hyperlink{classCollidable}{Collidable}} class inherits from the \mbox{\hyperlink{classUpdatable}{Updatable}} class and bundles all updatable objects that are to handle collisions. These objects have a circular collision shape.
