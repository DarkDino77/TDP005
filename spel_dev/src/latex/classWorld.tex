\doxysubsection{Detailed Description}
Represents the game-\/world. 

The \mbox{\hyperlink{classWorld}{World}} class is responsible for storing all game\+\_\+objects that are on the map. It provides the functionality to add new game objects and remove existing. The class also allows the game world to be simulated via ticks, using delta\+\_\+time
