\doxysubsection{Detailed Description}
An ammo pickup in the game that inherits from the \mbox{\hyperlink{classPick__Up}{Pick\+\_\+\+Up}} class. 

The \mbox{\hyperlink{classAmmo}{Ammo}} class inherits from the Pick-\/\+Up class and specifies what happens when the player picks it up. Also manages ammo-\/type.
